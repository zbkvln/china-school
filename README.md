# ТЯНЬ-ЛУН — Онлайн школа китайского языка

![ТЯНЬ-ЛУН](https://zbkvln.github.io/china-school/assets/dragon2.png)

**Современная онлайн-школа китайского языка** с авторской методикой.  
От постановки тонов до свободного общения.

## О проекте

- **Сайт**: [https://zbkvln.github.io/china-school/](https://zbkvln.github.io/china-school/)
- **Технологии**: HTML, CSS, JavaScript (чисто фронтенд)
- **Цель**: Привлечение учеников на курсы китайского языка
